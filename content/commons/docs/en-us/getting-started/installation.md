# Installation

This guide explains how to install TransitCore and prepare your development environment to create your own Addons.

At the end of this page, you will have:

- TransitCore installed.
- A functional Minecraft environment.
- LuaTC support.
- A project ready to host your first Addon.

<Alert severity="info">

TransitCore is currently developed for **NeoForge** and requires a compatible version of Minecraft. Always check that your game version matches the one supported by TransitCore.

</Alert>

## Prerequisites

Before installing TransitCore, make sure your computer has the following software.

| Software | Minimum required | Recommended |
|----------|----------------|------------|
| Java | Version 21 or higher | Latest LTS version |
| Minecraft | Compatible version | Latest supported version |
| NeoForge | Compatible version | Latest supported version |
| Git | Optional | Latest version |
| Visual Studio Code or IntelliJ IDEA | Optional | Latest version |

## Install TransitCore

Download the latest version of TransitCore.

Then place the mod file in the `mods` folder of your Minecraft installation.

```text title="Minecraft folder"
.minecraft/
├── mods/
│   └── transitcore.jar
└── ...
```

Launch Minecraft a first time to allow TransitCore to automatically create the folders it needs.

<Alert severity="success">

On first launch, TransitCore automatically generates all the folders necessary for its operation.

</Alert>

## Generated folders

After its first launch, TransitCore creates several folders in your Minecraft directory.

```text title="TransitCore folders"
.minecraft/
├── TC_Addons/
├── TC_Cache/
├── TC_Logs/
├── TC_Config/
└── mods/
```

| Folder | Description |
|---------|-------------|
| `TC_Addons` | Contains all installed Addons. |
| `TC_Cache` | Stores cache files generated by the framework. |
| `TC_Logs` | Contains TransitCore logs. |
| `TC_Config` | Contains the framework configuration files. |

## Install LuaTC support

TransitCore provides official tools to facilitate development in LuaTC.

The currently supported editors are:

- Visual Studio Code
- IntelliJ IDEA

The official extensions notably provide:

- Syntax highlighting.
- Auto-completion.
- Diagnostics.
- Integrated documentation.
- Code navigation.
- Automatic formatting.

<Alert severity="info">

LuaTC extensions are optional, but highly recommended to improve your development comfort.

</Alert>

## Verify the installation

Launch Minecraft.

If TransitCore is correctly installed, you should see a message similar to the following in the console.

```text title="Console output"
[TransitCore] TransitCore successfully initialized.
[TransitCore] Loading addons...
[TransitCore] Ready.
```

In case of a problem, consult the logs generated by TransitCore.

```text title="Logs folder"
.minecraft/
└── TC_Logs/
```

## Update TransitCore

To install a new version of TransitCore:

1. Close Minecraft.
2. Replace the existing JAR file with the new version.
3. Relaunch Minecraft.

When possible, TransitCore automatically migrates compatible configuration files.

<Alert severity="warning">

Before any update, it is strongly recommended to back up your world, especially if you are using an Alpha or Beta version.

</Alert>

## Uninstall TransitCore

To completely remove TransitCore:

1. Delete the TransitCore JAR file from the `mods` folder.
2. Delete the Addons you no longer need in `TC_Addons`.
3. If you wish, also delete the folders generated by TransitCore.

```text title="Folders that can be deleted"
TC_Addons/
TC_Cache/
TC_Config/
TC_Logs/
```

Deleting these folders will permanently erase installed Addons, configuration files and cached data.
